# 医院管理系统 java668

## 项目概述

医院管理系统是基于前后端分离架构开发的计算机项目程序设计管理系统。该系统结合了Java技术与现代Web技术，致力于为医院提供高效、易用、稳定的管理解决方案。

## 技术栈

- **后端技术：**
  - Java
  - Spring
  - Spring Boot
  - MVC模式
  - MyBatis
  - MySQL
  - Maven
- **前端技术：**
  - HTML
  - JavaScript
  - CSS
  - Vue.js
- **开发工具：**
  - IntelliJ IDEA
  - VS Code
- **运行环境：**
  - JDK 1.8
  - MySQL数据库
  - Redis
  - Node.js 14

## 功能模块

系统包含但不限于以下功能模块：

- 患者管理
- 医生排班
- 药品库存管理
- 费用结算
- 系统用户权限管理

## 系统角色

- 系统管理员
- 医生
- 药剂师
- 挂号收费员

## 运行环境

- 服务器端：JDK 1.8 或更高版本，MySQL数据库
- 客户端：现代Web浏览器

## 部署步骤

1. 环境准备：
   - 安装JDK 1.8
   - 安装MySQL数据库
   - 安装Redis
   - 安装Node.js 14
   - 安装IntelliJ IDEA和VS Code
   - 安装Maven
2. 数据库配置：
   - 使用navicat或其他数据库管理工具创建数据库，并根据文档执行SQL脚本。
3. 后端部署：
   - 使用IntelliJ IDEA打开项目，配置好数据库连接，然后编译打包。
   - 部署至服务器。
4. 前端部署：
   - 使用VS Code打开前端项目，编译打包。
   - 将打包好的文件部署至Web服务器。
